# Multiplayer Snake 🐍

## Play Locally
```bash
npm install
npm start
```
Open http://localhost:3001

## Features
- **Multiplayer Snake Race**: Shared food, snake collision, live scores
- WASD/Arrows controls
- Real-time sync via Socket.io
- Leaderboard persistence
- Responsive design

## Deploy to Render.com
1. Push to GitHub
2. Render.com → New → Web Service → Connect repo
3. Node runtime → Deploy!

## How to Play
1. Enter username
2. Quick Play "Snake" or Create Room "Snake"
3. Invite friends (same room)
4. Host clicks **Start Game**
5. **Eat food first**, avoid collision!
6. Dead snakes gray out 🟢→💀

**Multiple players = more fun!**

Live demo: [Deploy your own!](https://render.com)
